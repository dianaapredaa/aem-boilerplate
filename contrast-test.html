Color Contrast Accessibility Test

This page demonstrates good and poor color contrast for accessibility testing.

Good Contrast (Normal Content)

This text has good contrast and should be easily readable. The text color is #131313 on a white background, providing excellent readability.

Normal Button

Severe Contrast Issue #1

This text is nearly invisible! White text (#f5f5f5) on pure white background (#ffffff) creates an extremely poor contrast ratio of approximately 1.02:1, far below WCAG requirements.

Users with visual impairments, color blindness, or those viewing in bright sunlight will have significant difficulty reading this content.

Invisible Button

This link is nearly invisible due to severe contrast failure.

Severe Contrast Issue #2

White text on bright yellow background - this fails WCAG contrast requirements and is hard to read!

Severe Contrast Issue #3

Light red text on red background - another severe accessibility violation that makes text nearly unreadable.

Instructions for Testing

Use accessibility tools like:

  • Browser DevTools Accessibility panel
  • WAVE Web Accessibility Evaluator
  • axe DevTools extension
  • Lighthouse accessibility audit

These tools should flag the content in the gray box above as having insufficient color contrast.
